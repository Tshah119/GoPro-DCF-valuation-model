GoPro DCF Valuation Model

Overview
This project develops a Discounted Cash Flow (DCF) valuation model to estimate the intrinsic value of GoPro, Inc. The model forecasts future free cash flows and discounts them to present value using the Weighted Average Cost of Capital (WACC).

Objective
The goal of this model is to analyze GoPro’s financial performance and estimate its intrinsic valuation under different assumptions using financial forecasting and valuation techniques.

Model Components
• Revenue forecasting
• Operating margin projections
• Free cash flow calculations
• Weighted Average Cost of Capital (WACC) estimation
• Terminal value calculation using the Gordon Growth Method
• Sensitivity analysis
• Scenario analysis (Base, Bull, Bear cases)

Interpretation
Based on the assumptions used in the base-case scenario, the model estimates GoPro’s intrinsic value and evaluates whether the company appears undervalued or overvalued relative to its market price.

Tools Used
Microsoft Excel

Purpose
This model was created as an academic project to demonstrate financial modeling and valuation skills commonly used in corporate finance, equity research, and investment analysis.
